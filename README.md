# 🤖 Basic Chatbot using Python
A simple rule-based chatbot developed using Python. The chatbot interacts with users through predefined responses based on their input

---

## ✨ Features
- Greets the user
- Responds to basic questions
- Handles unknown inputs
- Exits the chat when the user types `bye`
- Simple and beginner-friendly

---

## 🛠️ Technologies Used
- Python 3
- VS Code (or any Python IDE)

---

## 📚 Python Concepts Used
- Functions
- While Loop
- If-Else Statements
- User Input
- String Methods

---

## ▶️ How to Run

```bash
python chatbot.py
```

---

## 💬 Sample Conversation

```
You: hello
Bot: Hi! Nice to meet you.

You: how are you
Bot: I'm fine, thank you!

You: what is your name
Bot: My name is CodeAlpha ChatBot.

You: bye
Bot: Goodbye! Have a nice day.
```
